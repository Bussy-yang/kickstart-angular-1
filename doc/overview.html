Kickstart for Appweb

Kickstart for Appweb is a fully working sample application that you can use as a starting point for your management web application. Because Kickstart uses proven designs and includes most of the required elements for a secure web aplication, you will lower your project risk and cut months off your development schedule.

Development Risks

- HTML5 - Browser compatibility - Security - User Login - Visualizations with live data

Compatibility

The Kickstart application sample is intended to be a starting point for a custom application. As such, it will not provide any assurances regarding API stability or compatibility for future releases. It is not anticipated or supported that customers will update their applications for future editions of Kickstart.
