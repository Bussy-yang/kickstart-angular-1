Directive: esp-modal

<esp-modal ...>
Synopsis
<esp-modal> Body</esp-modal>
Description

The esp-modal directive wraps any set of HTML elements and renders in a modal dialog.

Attributes
None
Example
<esp-modal> Body </esp-modal>
