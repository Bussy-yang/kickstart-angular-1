Directive: esp-confirm

<esp-confirm ...>
Synopsis
<esp-confirm header="Header" body="Body?" ok="Button">
Description

The esp-confirm directive displays a modal dialog and awaits user input.

Attributes
body: String Confirmation dialog body text.
header: String Dialog header text.
ok: String Dialog accept button text.
Example
<esp-confirm header="Are you sure?" body="Do you want to remove user: {{user.username}}?" ok="Delete User">
