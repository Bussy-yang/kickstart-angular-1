Directive: esp-gauge

<esp-gauge ...>
Synopsis
<esp-gauge value="{{sales.total}}" title="Total Sales" range="0-1000000">
Description

The esp-gauge directive renders numeric data in the form of a simple gauge. If the value to display is data bound and is updated, the gauge will also be updated.

The gauge may be configured via HTML attributes or via similar named properties on the $scope. Style defaults are set by the .gauge style for font-size, background and foreground colors.

Attributes
color: String Text and arrow color.
height: String Gauge height in pixels.
highlight: String Background color of the gauge sweep area.
range: String Min-Max values.
title: String Gauge title.
value: String Gauge current value to display.
width: String Gauge width in pixels.
Example
<esp-gauge value="{{ports.active}}" title="Active Ports" range='0-16'></esp-gauge>
